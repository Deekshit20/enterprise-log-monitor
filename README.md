# Enterprise Log Monitoring System (SQL)

## Overview
A SQL-based system to monitor logs, detect errors, and generate alerts in real-time.

## Features
- Log storage and indexing
- Error detection and tracking
- Automated alerts using triggers
- Reporting using views
- Stored procedures for querying

## Tech Stack
- SQL (MySQL)

## Use Case
Used in enterprise systems to monitor application logs and detect failures.

## Highlights
- Real-time alert system (Triggers)
- Performance optimization (Indexes)
- Modular SQL design
